# design-patterns

A repository that contains examples of the following Design Patterns:
 1. Strategy Pattern
 2. Observer Pattern
 3. Decorator Pattern
 4. Factory Pattern
 5. Singleton Pattern
 6. Command Pattern
 7. Adapter and Facade Patterns
 8. Template Method Pattern
 9. Iterator and Composite Patterns
10. State Pattern
11. Proxy Pattern
12. Compound Patterns
